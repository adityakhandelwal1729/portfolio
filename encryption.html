AES-CTR Encryption Core: Secure Root of Trust RTL-to-Signoff Flow

I implemented a Register Transfer Level-to-Graphic Data System II (RTL-to-GDSII) flow for a secure Root of Trust (RoT)CPU) using Verilog. The design integrates a pipelined 128-bit Advanced Encryption Standard in Counter Mode (AES-CTR) engine, a Physically Unclonable Function (PUF), and a Ring Oscillator–based True Random Number Generator (TRNG) as part of the secure core. Automatic Place and Route (APR) was executed using the FLAT flow in Cadence Innovus, with verification of both timing and power intent.

I drove the design flow using Cadence Genus for logic synthesis and Innovus for Place and Route (PNR) with the 7-nanometer ASAP7 Process Design Kit (PDK). Comprehensive timing and power signoff were performed through Synopsys Design Constraints (SDC) and Common Power Format (CPF), optimizing the design for approximately 1 gigahertz (~1 GHz) operation and low power consumption.
